Add, Edit and Delete Goals


Users would be able to add, edit or delete any goals as necessary. Fitness, Nutrition and Healthy Lifestyle goals can be created and viewed on their specific pages.




Achieving Goals


Once set goals have been achieved, users can complete them. All completed goals can also be viewed, which also shows what date the goals were completed.




ApplicationPages



Fitness



On this page the user is able to create, edit, delete and complete fitness goals. Also, the user is able to view the following information, to aid in creating goals:

  • Setting Goals
  • Creating a Plan
  • Benefits of Exercise
  • Types of Exercises
  • Safety Tips

Nutrition



On this page the user is able to create, edit, delete and complete nutrition goals. Also, the user is able to view the following information, to aid in creating goals:

  • Fueling for Exercise
  • Timing of Meals
  • Hydration
  • Macronutrients
  • Supplements

Healthy Lifestyle



On this page the user is able to create, edit, delete and complete healthy lifestyle goals. Also, the user is able to view the following information, to aid in creating goals:

  • Regular Exercise
  • Physical Activity
  • Mind-Body Exercises
  • Outdoor Exercise
  • Active Lifestyle
